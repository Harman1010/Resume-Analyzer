## Resume Analyzer

This project is about providing real time ATS score and providing suggestions based on the ATS Score.

##Working

The project follows hybrid structure where we used pre-trained state-of-the-art models for semantic understanding along with system design approach
to combine and develop a formula for calculating ATS score and then using template-based code to generate suggestions.

##Tech Stack

Python
SentenceTransformer
KeyBERT
PDF Plumber
